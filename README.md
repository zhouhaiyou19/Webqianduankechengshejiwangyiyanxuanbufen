## 项目简介

本资源文件是本科大三上学期Web基础课程设计的一部分，主题为“网易严选”。由于是小组作业，项目链接仅包含我负责的商品详情页面和部分首页内容。文档由我本人撰写，详细介绍了项目的实现过程和相关技术。
